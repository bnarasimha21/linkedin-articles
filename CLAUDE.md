### Style Guidelines:
1. Keep paragraphs SHORT (2-3 sentences max)
2. Use line breaks between paragraphs
3. Use â€” (em dash) as section dividers

5. Use â†’ for lists instead of bullets when showing flow

7. End with 3-6 relevant hashtags

9. Write conversationally, not formally

### File Naming:
- Source: `XX-article-name.md` (full markdown version)
- LinkedIn: `XX-article-name-linkedin-ready.txt` (copy-paste ready)

### Workflow:
1. Write article in markdown (for GitHub/blog)
2. Create `-linkedin-ready.txt` version with formatting above
3. User can copy-paste directly to LinkedIn
